# i3 dotfiles

i3 dotfiles with nord theme. 

![Screenshot](/images/screenshot.png)

## Installation

```bash
git clone https://github.com/zvdy/i3-dotfiles
cd i3-dotfiles
```

### Run the install script

```bash
python3 install.py
```

### Dependencies

- i3
- polybar
- rofi
- zsh
- oh-my-zsh
- powerlevel10k
- nerd-fonts
- feh
- neofetch
- kitty
